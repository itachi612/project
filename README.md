# UNIX Data Processing Pipeline (CLI)

## О проекте

Небольшой проект, в котором я реализовал обработку данных с использованием UNIX-инструментов без Python.  
Основная идея — пройти полный путь работы с данными через командную строку: от получения данных до базового анализа.

В качестве источника использовалось API вакансий, после чего данные проходили несколько этапов обработки.

---

## Что реализовано

Проект покрывает базовый pipeline работы с данными:

- получение данных из API (`curl`)
- работа с JSON (`jq`)
- преобразование данных в CSV
- сортировка и очистка данных
- извлечение полезной информации из текста
- подсчёт и анализ распределений
- разбиение данных на части и обратная сборка

---

## Стек

- Bash / Shell scripting  
- UNIX utilities (`curl`, `jq`, `sort`, `uniq`, `sed`, `cat`)  
- CSV / JSON  

---

## Структура

- src/
  - ex00/ — загрузка данных  
  - ex01/ — JSON → CSV  
  - ex02/ — сортировка  
  - ex03/ — очистка  
  - ex04/ — статистика  
  - ex05/ — разбиение и объединение  


---

## Примеры задач

**1. Получение данных**  
Скрипт делает запрос к API и сохраняет вакансии в JSON.

**2. Преобразование формата**  
JSON преобразуется в CSV с нужными колонками для анализа.

**3. Очистка данных**  
Из названий вакансий извлекается уровень:
- Junior
- Middle
- Senior
- или "-" если уровень не определён

**4. Анализ**  
Подсчитывается количество каждой категории.

**5. Работа с данными**  
Файл разбивается по датам и собирается обратно.
